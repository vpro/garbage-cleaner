Purges a list of folders. Deletes all files not accessed for a specific time period.

(Used timing in minutes)

Usage:

[source]
----
    docker run \
        --env "TARGET_FOLDERS=cache,/assets" \
        --env "FILE_AGE=2" \
        --env "CRON=*/5 * * * *" \
        garbage-cleaner
----

Also, it will clean up logs in /data/logs
